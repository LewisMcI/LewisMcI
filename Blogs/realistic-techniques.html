Goal

The inspiration for this project was from a viral video of the game 'Unrecord'. This game uses multiple realistic techniques to simulate body cam footage.

Image From: Unrecord Steam Page

A teammate and I from a previous project were interested in the techniques used to create a realistic Aim Down Sight (ADS). Movement delay, fish eye distortion, camera lense effect.
