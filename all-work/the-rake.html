The Rake

September 20th, 2023


Quick Navigation

Description

Horror Game template for Unity. Intended as setup for an eventual Horror game. Featured many mechanics found within generic Horror games.

Within this project, I tried to optimise my own workflow, by finding and gaining experience in all parts of Unity. This included the UI Toolkit, Decal System, Lighting System and Bounding Volumes.


Features

  • Dynamic Music Manager with Smooth Transitions
  • Component Based Interaction System
  • Positional Decal System
  • Jumpscare Manager
  • Draggable Inventory System
  • Audio Clip Parser for Noise Meter
