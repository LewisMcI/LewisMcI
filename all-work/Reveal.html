Description

The idea for this project, which was 'Light Reveals', came from a previous group project. I have repurposed it and make a tech demo showcasing the shader. The initial shader was written in HLSL, and then converted into Shader Graph such that it could utilize the new features found in the Universal Render Pipeline.
