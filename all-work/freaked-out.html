Description

Freaked Out is a Singleplayer, Comedic, First person Simulation and Sandbox game following the brief 'Dumb Ways to Die'. It centers around a character trapped in a room by their alter-ego, forced to fight through deadly traps which gets significantly more devious as the days go by.


My Contribution

The focus of this project was developing my team-based skills. This is shown through my designer-friendly coding practices, giving easy-to-use tools that help balance gameplay.

  • Unity & GitHub Management
  • AI FSM with NavMesh Pathfinding, Procedural Animation & Line of Sight mesh creator
  • Designer-Friendly Custom UI System using Delegates
  • Simple Player Game Settings
  • AI Assisted Smooth Procedural Animation
  • Easy Setup Dialogue System
  • Scripted Object Task Generation
  • Dynamic Camera System
  • Stylized VFX Creation
  • Misc (Including major debugging & QoL adjustments)
